# whoport
A simple CLI tool to check if and which PID uses a given port number and optionally kill it.

**Tested on OSX 10.10.5, 10.11.4 and Window 7**

PRs welcome!

### Usage

```
# check and display PID if present
whoport 9090

# kill process which uses port
whoport 9090 -k
```
